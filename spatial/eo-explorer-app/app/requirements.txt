# Deploy manifest for Streamlit Community Cloud.
#
# Cloud searches the entrypoint file's directory first, then the repo root, so
# this file (next to main.py) is the one it installs from. Pick Python 3.12 in
# the deploy dialog's Advanced settings: eo-monitor requires Python < 3.13.
#
# eo-monitor is installed from this same public repo (the eo-monitor/ folder),
# so the app uses one definition of the spectral indices instead of a copy.
#
# The git ref is pinned to a specific commit (not @main) on purpose: Streamlit
# Community Cloud caches the git install, and a moving @main ref can leave an
# older eo-monitor in place after the library gains new indices. Pinning a
# commit makes the requirement string change whenever eo-monitor does, which
# forces a clean reinstall. Bump this SHA when you want the app to pick up newer
# eo-monitor changes.
